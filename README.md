# W3CPL-World-Wide-Web-Comment-Policy-License-

W3CPL (World Wide Web Comment Policy License) is a lightweight,  
reusable comment policy standard designed for use across a wide range of  
online spaces on the World Wide Web, including blogs, SNS, forums,  
chat systems, and community platforms.

This repository provides the official license text, usage guidelines,  
and version history.
